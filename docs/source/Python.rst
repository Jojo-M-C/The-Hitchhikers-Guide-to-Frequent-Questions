Python 
=======

Keywords 
---------

**True, False**

Data values from the data type Boolean

.. code:: yaml
    False == (1 > 2), True == (2 > 1)

**and, or, not **

Logical operators:

(x and y) → both x and y must be True

(x or y) → either x or y must be True

(not x) → x must be false

.. code:: yaml
    x, y = True, False
    (x or y) == True # True
    (x and y) == False # True
    (not y) == True # True
